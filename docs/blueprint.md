# **App Name**: StreetVerse

## Core Features:

- Hyperlocal Ordering: Enable users to order from nearby grocery stores, medical stores, and other local businesses.
- Mobile Doctor Booking: Allow users to book mobile doctors, select locations for their arrival, and assign them to users (similar to ride-hailing).
- Delivery Assignment & Tracking: Assign delivery tasks to 'Saviours,' track their location via Google Maps and GPS, and use OTP verification to confirm deliveries.
- Integrated Payments: Offer multiple payment options: cash on delivery, UPI (integrating with apps like Google Pay and PhonePe), and direct banking apps integration.
- Medicine Search & Booking: Enable users to search for medicines display result from medicine  database, add them to the cart, specify quantities, and book orders for delivery.
- Grocery Ordering & Tracking: Allow users to search for grocery products from grocery database, confirm orders, assign delivery personnel, and track orders via Google Maps and GPS.
- Job Board & Updates: AI-powered "tool" which posts the local and national job vacancies by updated database of jobs which is updated in "For Saviour " section
- Offline Access: Allow users to access the app in online and offline mode.

## Style Guidelines:

- Primary color: Light blue (#ADD8E6), reminiscent of the sky, promoting trust and reliability.
- Background color: Off-white (#F0F8FF), provides a clean and calm base.
- Accent color: A slightly darker blue (#77B5FE) is applied to calls to action, drawing user attention without being overly aggressive.
- Body and headline font: 'PT Sans', sans-serif for clarity and modern appeal, ideal for app readability.
- Consistent use of clear, minimalist icons in a uniform style for easy navigation and quick understanding of functions.
- Vertical menu sections for key functions (Mobile Doctor, Medicines, Groceries, Job Finder), with a horizontal 'For People' and 'For Saviours' menu at the top.
- Subtle animations (e.g., loading spinners, transitions) to improve the user experience and provide feedback on interactions.
